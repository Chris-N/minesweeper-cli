# minesweeper-cli
Simple Minesweeper game easily play through your console

Naviagte to the installed directory.

```
node game.js
```

Game Rules:

Letter represent a column
Number represent a row

```
   A B C D E F G
1	_|_|_|_|_|_|_
2	_|_|_|_|_|_|_
3	_|_|_|_|_|_|_
4	_|_|_|_|_|_|_
5  | | | | | | 
```

Commands
-select a square: C3
-flag a square: C3x

Have fun!
